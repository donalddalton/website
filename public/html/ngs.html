Background
High-throughput biological data typically has two primary sources of variance:
[1] Technical/Sampling: Sample preparation i.e. concentrations, reaction duration, batch etc.
[2] Biological: Sample heteroge nity, mRNA content changes within cells, donor (age, race, sex) etc.
The goal of high throughput experimental des ign is to minimize technical variability and account for sources
of biological variablity in your statistical model that do not contribute meaningful information to the question
you are asking. Data pre-processing provides methods for minimizing technical variability and informs
potential sources of biological variability that must be acc ounted for in downstream analysis.

The limma algorithm uses an approach called mixed-eects linear modeling to analyze mRNA expression data.
The approach requires one or two matrices to be specied. The rst is the design matrix which indicates in
eect which RNA samples have been applied to each array. The second is the contrast matrix which species
which comparisons you would like to make be tween the RNA samples. The philosophy of the approach is as
follows: You have to start by tting alinear model to your data which fully models the systematic (xed)
part of your data. The model is specied by the design matrix. Each row of the design matrix corresponds
to an array in your experiment and each column corresponds to a coecient that is used to describe the
xed eects of your data. The main purpose of this step is to estimate the variability in the data, hence the
systematic part needs to be modeled so it can be distinguished from random variation. For statistical analysis
and assessing dierential expression, limma uses an empirical Bayes method to moderate the standard errors
of the estimated log-fold changes for given contrasts. The general workow is as follows:

Principle Coordinate Analysis (PCoA)
Form of multidimensional scaling analogous to principle component analysis. This algorithm provides visual
representations of the similarity between a set of object (samples) based upon the euclidean distance between
its components (genes). The distance between objects can therefore be thought of as negatively directly
proportional to the similarity between them. Another way of thinking about MDS plots is that the distance
between objects is equivalent to the fold-change between them.

MA-plots
MA-plots are useful for visualizing reproducibility between samples for a given experiment. MA-plots are
constructed by plotting the log ratio of gene counts between samples against the log-average of gene counts
across samples. We expect samples with similar gene expression to have data points lying primarily along the
y=0 axis. Since MA-plots represent comparisons between 2 samples, it is useful to compute and visualize
several comparisons and look to see if the trends match our expectations about the data.
