Abstract
Advances in high throughput biological chip assays have led to vast amounts of biological
data. SNP arrays are a particular class of microarray that are used to detect single nucleotide
changes in the genome. SNP’s are sites on DNA that vary amongst a population of individuals
by more than 1% and have demonstrated predictive power with respect to disease association
and susceptibility [1]. A 2015 study showed that total sequence data had been doubling at a
7 month rate since 2009, a ﬁgure on the order of zetabytes when projected to 2025 [2]. The
extraction of meaningful information from such a large input domain is a natural application
of unsupervised machine learning. Current methods for cluster analysis of SNP data are
either slow due to a massive feature space, or are limited in the information that they can
extract by relying soley on linear transformations of the data. As such, we propose that
the use of unsupervised deep learning algorithms provide an opportunity to address both of
these issues by learning a compressed latent representation of the data that captures robust
non-linear relationships amongst features. Previous studies have shown that deep learning
models such as stacked autoencoders and deep belief networks can provide state of the art
results with respect to data compression [3]. We present results of the application of these
types of models to the task of compressing the SNP proﬁles of a pool of 5,000 indivuals
that represent 1,500 unique families. We then assessed the quality of learned features with a
generic nearest neighbors classiﬁer.
1 Introduction
1.1 Background & Related Work
Advances in high throughput biological chip assays have led to vast amounts of biological data. SNP arrays
are a particular class of microarray that are used to detect single nucleotide changes in the genome of which
roughly 99.9 % is identical amongst individuals. SNP’s are sites on DNA that vary amongst a population of
individuals by more than 1% and have demonstrated predictive power with respect to disease association and
susceptibility [1]. Currently, there are a number of chip technologies that screen for thousands of SNP’s and
provide a rich, high-dimensional description of a persons genetic proﬁle. This technology has experienced a
great deal of success and a 2015 study showed that total sequence data had been doubling at a 7 month rate
since 2009, a ﬁgure on the order of zetabytes when projected to 2025 [2]. One major application SNP data
studies the relationship between a persons SNP proﬁle and phenotype. In large scale genome wide association
studies (GWAS), unsupervised learning methods can provide insight into the latent structure of SNP proﬁles
and their relationship to population phenotypes. Almost all current clustering methods employed on SNP
datasets utilize some form of distance-based metric to calculate similarity amongst a pool of proﬁles [4]. These
methods are slow in practice, and do not scale well with the size of the dataset. Further, typical biological
datasets are comprised of measurements of continuous variables whereas SNP data is discrete in nature. For
this reason there have been considerably fewer studies on clustering methods for SNP datasets. Currently, the
k-modes algorithm is one of the most popular choices for clustering of large discrete datasets [5]. This is an
extension of the classic k-means algorithm in which optimal cluster modes are identiﬁed via a frequency-based
method. The same drawbacks of the k-means algorithm apply to k-modes, namely a non-convex objective
function whose optimization is NP-hard for k > 1. For a given dataset with

As a result, most real world applications of k-NN require some form of dimensionality reduction such as
PCA in order to shrink the feature space. Dimensionality reduction techniques such as PCA apply linear
transformations to the data. More speciﬁcally, PCA attempts to identify a set of orthogonal vectors of lower
dimensionality that minimize reconstruction error of the data when it is projected into this lower dimensional
space. The vector that minimizes reconstruction error is given by the eigenvector of the covariance matrix
Σ
with largest eigenvalue. There are a number of methods for ﬁnding these eigenvectors, namely power iteration,
singular value decomposition (SVD) aka latent semantic analysis (LSA) in the context of NLP related tasks,
and many other approximate algorithms. All of these methods rely on linear transformations of the data and
are thus limited in the information that they can extract. This is problematic when dealing with a large set of
features that have complex non-linear relationships, as is the case with SNP datasets. The number of features
in a typical SNP dataset is on the order of thousands, however this number continues to increase as these
technologies improve. Thermoﬁsher has recently begun to offer an SNP array that screens over 1.8 million
SNP’s. Clearly there is a need for better dimensionality reduction algorithms that can capture the complex
structure of this data and drastically improve runtime with respect to classiﬁcation/retrieval tasks.
1.2 Deep Generative Model
We have addressed the issue of efﬁcient cluster analyis in a manner inspired by the semantic hashing algorithm
[4], which was developed to address the problem of fast document retrieval with a deep generative graphical
model. This method allows for retrieval of similar documents in time independent of document length and linear
in the number of neighbors to return for a given query. Speciﬁcally, this method consists of training a deep
belief network consisting of multiple layers of stochastic binary latent variables. The network is "pretrained"
via a recursive greedy approach in which each layer can be viewed as a single layer restricted roltzman machine
(RBM) trained via the Constrastive Divergence algorithm [6]. Upon completion of pretraining the model is
unrolled into a stacked autoencoder and ﬁne-tuned via gradient descent. When the number of hidden units
is less than the dimensionality of the input, these models can learn robust low dimensional representations
of the data. These models have demonstrated state of the art performance at NLP tasks such as document
classiﬁcation and data compression [4]. We have applied this framework to the task of learning a compressed
representation of SNP data. Speciﬁcally, we have constructed a deep generative model (DGM) in which the
lowest (visible) layer represents a high dimensional binary SNP expression vector, and the top layer represents
a low dimensional, learned binary representation over a given expression proﬁle. The top two layers form an
undirected network while the remaining layers have directed, bottom down connections, ﬁgure 1. We can use
the features learned by this model to transform SNP expression proﬁles into a low dimensional representation
which can be used for classiﬁcation/retrieval tasks.

2 Methods
2.1 Description of Data
The majority of the largest SNP databases such as The Cancer Genome Atlas (TCGA), the 1000 Genomes
Project, and the Genome-Wide Repository of Associations Between SNPs and Phenotypes (GRASP) are tightly
regulated due to patient data privacy laws which limited the size of the training set we were able to obtain.
We were however able to obtain a reasonably large dataset from the Autism Genome Project (AGP), which
collected SNP proﬁles from 4,878 individuals representing 1,496 unique families in a large multiplex study of
autism, ﬁgure 2. Selected families were those in which 2 or more members were affected and genomic samples
were collected from each family member. For each sample, the following metadata annotations are available:
family identiﬁer, disease status (affected/unaffected), gender (M/F), and relationship to affected individuals
(parent/sibling). While this is not an ideal dataset with respect to size, this sample still has many desirable
characteristics such as detailed metadata. Additionaly, the sampling strategies employed by large GWAS
studies are designed to minimize systematic bias and ensure sufﬁcient statistical power for distinguishing
potentially interesting signals from random chance [7]. We therefore believe this dataset provides a unique
opportunity to demonstrate ability of a deep generative model to learn a latent representation of the genome.

2.4 Training
The processed dataset consists of 4,878 SNP expression proﬁles each containing 18,665 binary features. Due
the lack of more training examples and an overdetermined dataset, we opted to not waste data by creating a
validation set. Instead hyperparameter tuning was done soley based on performance on the training set and
strategies to prevent overﬁtting were used during ﬁne tuning. 70% of the data was allocated to a training set
and the remaining 30% to the test set. Due to the variable number of individuals in each family, we made sure
that at most 70% of the individuals from a given family were allocated to the training set and for the small
number of families with only several members we made sure that no less than a third of them were allocated
to the test set. We then constructed the DGM in Figure 1 and trained each layer uing the greedy layerwise
approach for 30 epochs or until early stopping was achieved. Each RBM layer was trained to minimize cross
4
entropy reconstruction loss via the contrastive divergence algorithm. Next, the model was unrolled into a
3 layer stacked autoencoder and initialized with the weights learned during pretraining. We then trained
the stacked autoencoder for 30 epochs or until early stopping was achieved. The stacked autoencoder was
trained to minimize mean squared reconstruction error. We tested a variety of architectures as well as different
combinations of linear or sigmoid activation functions at each hidden layer. In order to prevent overﬁtting we
utilized early stopping and used a small learning rate of 0.0001 for all parameter updates. Upon completion of
training, low dimensional representations of both the test and train sets were obtained by passing them through
the network and collecting them at the middle layer of the autoencoder. We, then constructed a generic k-NN
classiﬁer with a euclidean distance function and used it to predict the family label of each example in the
test set using
k = 5, 10, 20
. For each value of k we calculated precision and recall scores which were then
averaged by family size and combined into an aggregate score. We constructed several reference models to
benchmark the DGM against. In order to test different dimensionality reduction methods against the DGM we
used the following models: single layer autoencoder, stacked autoencoder, stacked denoising autoencoder,
and PCA via singular value decomposition (SVD). Using the same k-NN approach as before we assessed the
performance of each benchmark with consistent dimensionality across every model. We also performed k-NN
on the untransformed data with a Hamming distance function, recall this is a current standard [4]. Performance
metrics were collected for each model over 5 random test/train splits.

4 Discussion
In this study we assessed the performance of several unsupervised learning methods at the task of dimensionality
reduction for SNP data. We constructed a deep generative model based on the Semantic Hashing algorithm.
Using the greedy layerwise pretraining method, we pretrained the deep generative model and unrolled it
into a stacked autoencoder. Next, we ﬁne tuned the model via the backpropagation algorithm. We then
used the learned features of the model to transform our data into a low dimensional representation, and
assessed it’s performance at classiﬁcation/retrieval tasks with respect to accuracy and runtime. While we
found that our model did not perform as well as expected, we found that other neural nets such as stacked
denoising autoencoders gave performance consistent with current methods such as k-NN with Hamming
distance function, and offered substantial improvements in runtime. The poor performance of our model is
almost certainly a result of a lack of sufﬁcient data. As a result the network was not able to represent the true
data generating distribution and generalized poorly. Efforts to further develop this model must address this
issue. Going forward there are a number of unexplored areas that should be investigated with respect to the
features learned by the model. One such example would be extracting biologically relevant information such
as gene regulatory networks or linking certain features to observed phenotypes. We leave this as a task to be
revisited if/when a larger more diverse dataset becomes available for public use.
