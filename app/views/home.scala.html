Machine Learning

GANs for scRNA-Seq Analysis


Assessed the performance of several unsupervised learning methods at the task of dimensionality reduction for single cell RNA-Seq data. Methods include PCA, t-distributed stochastic neighbor embedding, and a relatively new framework based on Generative Adversarial Networks.


Unsupervised Deep Learning for SNP Compression


Studied the performance of state of the art unsupervised deep learning algorithms at the task of compressing high dimensional SNP datasets. Such algorithms included sparse autoencoders, deep Boltzman machines, and deep belief networks.


Building a Machine Learning Pipeline


Case study to give practical example of how to build and deploy a machine learning pipeline.
